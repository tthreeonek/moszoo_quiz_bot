# 🐾 Московский зоопарк: Тотемное животное – Telegram Bot (Бот-викторина для Московского зоопарка)


## 📖 О проекте

Московский зоопарк – один из старейших зоопарков Европы. Чтобы привлечь взрослую аудиторию к программе опеки, был создан этот бот.  
Пользователь проходит увлекательную викторину «Какое у вас тотемное животное?», а в конце получает:

- персональный результат с описанием характера;
- фото настоящего обитателя зоопарка;
- информацию о том, как стать опекуном именно этого животного;
- возможность поделиться результатом в соцсетях.

Бот полностью написан на **Python** с использованием библиотеки `python-telegram-bot` версии 20.7.

---

## 🚀 Функциональные возможности

- ✅ **Лёгкий запуск** – команда `/start` или кнопка «Начать викторину».
- ✅ **Дружелюбный интерфейс** – понятные сообщения с эмодзи и кнопками.
- ✅ **5 вопросов с вариантами ответов** – вопросы легко изменить в `questions_data.py`.
- ✅ **Балльная система подсчёта** – каждый ответ добавляет балл одному из 6 животных.
- ✅ **Тай-брейк** – при ничьей задаётся дополнительный вопрос только среди лидеров.
- ✅ **Персонализированный результат** – фото, описание тотемного животного.
- ✅ **Программа опеки** – блок с информацией, как помочь животным, со ссылкой на сайт.
- ✅ **Работа с изображениями** – отправка реальных фото животных (локальные файлы или URL).
- ✅ **Шеринг в соцсетях**:
  - **Twitter** – твит с готовым текстом и ссылкой на бота;
  - **ВКонтакте** – запись на стене;
  - **Telegram** – инлайн-шаринг.
- ✅ **Контакт с сотрудником** – кнопка пересылает результат администратору бота.
- ✅ **Обратная связь** – отзывы сохраняются в `feedback.log`.
- ✅ **Повторное прохождение** – кнопка «Попробовать ещё раз» сбрасывает всё.
- ✅ **Конфиденциальность** – собираются только необходимые данные (результаты викторины не хранятся долго).
- ✅ **Масштабируемость** – код готов к росту числа пользователей.
- ✅ **Защита от блокировок** – поддержка SOCKS5-прокси.

## 🔧 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/moszoo-quiz-bot.git
cd moszoo-quiz-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3. Получение токена бота
- Напишите @BotFather в Telegram.

- Создайте нового бота командой /newbot и следуйте инструкциям.

- Скопируйте полученный токен.

### 4. Определение вашего Telegram ID
Напишите @userinfobot, чтобы узнать свой числовой ID.

### 5. Настройка config.py
Откройте файл и заполните обязательные поля:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
ADMIN_CHAT_ID = 123456789          # ваш Telegram ID (число)
BOT_USERNAME = "@MyZooBot"        # юзернейм бота (без пробелов)
```
Опционально можно указать прокси:

```python
PROXY_URL = "socks5://127.0.0.1:10808"
GET_UPDATES_PROXY = "socks5://127.0.0.1:10808"
```
### 6. Подготовка изображений
В папку ```images/``` положите:

Логотип (любой горизонтальный цветной логотип из вашего набора). Переименуйте его в ```logo.jpg```.

Фотографии животных (имена строго соответствуют ключам в ```animals_data.py```):

```elephant.jpeg``` → Азиатский слон

```meerkat.jpeg``` → Сурикат

```flamingo.jpeg``` → Розовый фламинго

```manul.jpeg``` → Манул

```tiger.jpeg``` → Амурский тигр

```polar_bear.jpeg``` → Белый медведь

Если хотите использовать прямые ссылки из интернета, измените в ```animals_data.py``` поле ```image_path``` на URL. Бот автоматически определит, что это не локальный файл, и загрузит по URL.

### 7. Запуск бота
```bash
python main.py
```
В консоли должно появиться сообщение: Бот запущен.
Теперь можете открыть своего бота в Telegram и начать викторину командой ```/start```.

## 🌐 Работа с прокси (если Telegram заблокирован)
Если вы находитесь в регионе, где Telegram недоступен, настройте прокси в ```config.py```:

```python
PROXY_URL = "socks5://127.0.0.1:10808"
GET_UPDATES_PROXY = "socks5://127.0.0.1:10808"
```
Поддерживаются только SOCKS5-прокси (например, от Tor, Outline VPN или локального клиента).

## 🧩 Настройка викторины и животных
Изменение вопросов

Все вопросы находятся в ```questions_data.py```. Каждый вопрос имеет список вариантов с привязкой к ключу животного (```animal```).

Вы можете добавлять, удалять или редактировать вопросы. При изменении количества вопросов поправьте константу ```QUESTION_STATE``` в ```main.py``` (или оставьте ```range(len(QUESTIONS))```, она автоматически подстроится).

#### Добавление новых животных
1. Добавьте запись в словарь ```animals_info``` в ```animals_data.py```.

2. Добавьте ключ нового животного в существующие вопросы (или создайте новые).

3. Положите изображение в папку ```images/```.

4. Обновите инициализацию счётчиков в функции ```start_quiz``` (она использует ```animals_info.keys()```, поэтому новые ключи подхватятся автоматически).

#### Смена фотографий
Просто замените файлы в папке ```images/```, сохранив имена.
Если хотите использовать URL, замените в описании животного ```"image_path": local_image(...)``` на ```"image_path": "https://..."```.

## 📤 Поддержка социальных сетей
После получения результата пользователь видит кнопки:

***🐦 Поделиться в Twitter*** – открывает окно твиттера с готовым текстом «Моё тотемное животное — ...» и ссылкой на бота.

***💙 Поделиться ВКонтакте*** – публикация на стене ВКонтакте.

***🔄 Поделиться в Telegram*** – инлайн-режим, позволяющий отправить результат в любой чат.

Для корректной работы ссылок необходимо указать юзернейм бота в ```config.py``` (поле ```BOT_USERNAME```).
